## 背景

RSDP 需要在 Windows 本地服务器/工作站上部署，兼顾零运维成本、本地数据不出机器、以及后续可扩展性。

## 决策

- **Java 21 + Spring Boot 3**：企业级标准，虚拟线程适合并发调用 AI 推理服务。
- **PostgreSQL**：成熟的关系数据库，支持 JSON/JSONB、复杂查询、并发写入。
- **ChromaDB**：独立向量服务，REST API 调用，可替换为 Milvus/Qdrant。
- **Ollama / DashScope**：本地部署目标使用 Ollama 托管嵌入模型与 Qwen 2.5-VL 7B；MVP 阶段使用 DashScope `qwen3-vl-plus` 快速验证视觉识别链路。
- **Vue 3 + TypeScript**：现代前端技术栈，开发效率高。
- **MinIO**：S3 兼容对象存储，用于图片与导出文件。
- **Docker Compose**：一键编排六个服务，便于本地部署与迁移。

## 影响

- 本地部署无需购买云服务器。
- AI 推理可选择本地 NVIDIA 显卡或云端 API；无独显时本地 CPU 回退速度较慢。
