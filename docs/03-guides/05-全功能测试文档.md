# RSDP 全功能测试文档（前端操作版）

> **适用版本**：RSDP v1.0+
> **测试环境 URL**：http://localhost:5173（前端）/ http://localhost:8081（后端 API）
> **测试方式**：以前端页面操作为主（`[FE]`），辅助 API（`[API]`）和数据库查询（`[DB]`）
> **UI 框架**：Vue 3 + Naive UI

---

### 1.3 初始化步骤

#### 步骤 1：启动基础设施

```bash
# 进入项目根目录
cd E:\roomvip

# 启动数据库、Redis、ChromaDB、MinIO
docker-compose -f deploy/docker-compose.yml up -d postgres redis chromadb minio

# 验证容器运行状态
docker ps | grep -E "postgres|redis|chromadb|minio"
# 预期：4 个容器均为 Up 状态
```

#### 步骤 2：初始化数据库

```bash
# 设置环境变量（Windows PowerShell）
$env:PGPASSWORD="rsdp"

# 执行初始化脚本
Get-ChildItem database/V*.sql | Sort-Object Name | ForEach-Object {
    Write-Host "执行: $($_.Name)"
    psql -h localhost -p 5433 -U rsdp -d rsdp -f $_.FullName
}
```

**验证**：
```sql
-- [DB] 检查表数量（预期 50+ 张表）
SELECT count(*) FROM information_schema.tables WHERE table_schema='public';

-- [DB] 检查种子用户（预期 6 行）
SELECT username, nickname FROM sys_user;

-- [DB] 检查角色（预期 6 行）
SELECT role_code, role_name FROM sys_role;
```

#### 步骤 3：配置并启动后端

```bash
# 确认 application-dev.yml 或环境变量中已设置：
#   DASHSCOPE_API_KEY=sk-xxxxx
#   RSDP_ENCRYPTION_KEY=<32位十六进制密钥>
#   RSDP_JWT_SECRET=<随机字符串>

cd server
mvn spring-boot:run -Dspring-boot.run.profiles=dev

# 等待启动完成，观察日志无 ERROR
# 预期最后一行：Started RsdpApplication in XX.XXX seconds
```

**验证方式**（通过健康检查端点）：
- 在浏览器访问 `http://localhost:8081/actuator/health`
- 预期：页面显示 `{"status":"UP"}`

#### 步骤 4：启动前端

```bash
cd web
pnpm install
pnpm dev
# 等待 Vite 启动，默认 http://localhost:5173
```

**验证**：
- 打开浏览器，访问 `http://localhost:5173`
- 预期：页面正常加载，显示 RSDP 登录页面或首页（若未登录则自动跳转到 `/login`）


```sql
SELECT rsku_id, factory_price, LENGTH(factory_price::text) AS cipher_length

-- 查看方案-订单关联
SELECT s.scheme_name, do.order_no, do.status
FROM scheme s
LEFT JOIN design_order do ON s.scheme_id = do.scheme_id
WHERE s.status = 'active' AND s.deleted_at IS NULL
LIMIT 20;

-- 查看用户-角色关系
SELECT u.username, r.role_code
FROM sys_user u
JOIN sys_user_role ur ON u.user_id = ur.user_id
JOIN sys_role r ON ur.role_id = r.role_id
ORDER BY u.username;

-- 查看企业-成员关系
SELECT c.company_name, u.username, c.price_ratio
FROM company c
JOIN sys_user u ON u.company_id = c.company_id
WHERE c.deleted_at IS NULL;

SELECT username, LEFT(password_hash, 7) AS hash_prefix
```
