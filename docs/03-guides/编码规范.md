## Java

- 使用 Java 21 特性，优先使用 Record 作为 DTO。
- 使用 Lombok 减少样板代码。
- Service 层方法名使用动词开头，如 `createDraft`、`updateByAiResult`。
- JSON 字段在实体中用 `String`，统一通过 `JsonUtils` 序列化/反序列化。
- 价格等敏感字段使用 AES 加密 TypeHandler。

## RESTful API

- 统一前缀 `/api/v1`。
- 统一响应 `Result<T>`，分页响应 `PageResult<T>`。
- Controller 只负责参数校验和调用 Service。

## Vue 3 / TypeScript

- 使用 `<script setup lang="ts">`。
- 类型定义统一放 `src/types/`。
- API 调用统一放 `src/api/`。

## 数据库

- 表名使用小写下划线。
- JSON 字段在 PostgreSQL 中存 `TEXT`（后续可优化为 `JSONB`）。
- 所有修改 RSPU/RSKU/工厂的操作必须记录审计日志。
